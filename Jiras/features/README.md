# Features

This directory contains the business features developed for the Farm Management System.

A feature represents a significant piece of functionality that delivers business value and supports one or more user stories. Features are grouped under their corresponding Epic and describe the capabilities provided by the system.

Each feature includes:
- A description of the functionality.
- Business value.
- Scope.
- Acceptance criteria.
- Related user stories.
- A screenshot of the corresponding Jira issue.
