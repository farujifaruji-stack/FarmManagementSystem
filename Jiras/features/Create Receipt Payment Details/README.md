# Create Receipt Payment Details

## Overview

The Create Receipt Payment Details feature enables farm administrators to record payment information associated with a digital receipt. It supports multiple payment methods and stores the necessary payment details to ensure accurate financial tracking, reconciliation, and auditing.

---

## Business Value

Provides a standardized way to document how receipts were paid, improving financial transparency, supporting auditing requirements, and maintaining complete payment records for farm transactions.

---

## Scope

This feature allows administrators to:

- Record payment information for a receipt.
- Select the payment method.
- Store payment reference details.
- Record bank or check information when applicable.
- Associate payment details with a digital receipt.
- Save optional notes related to the payment.

---

## Payment Information

The following information can be recorded depending on the selected payment method:

- Payment Method
- Payment Reference Number
- Bank Name
- Branch Number
- Card Last 4 Digits
- Card Holder Name
- Check Number
- Check Date
- Check Bank Name
- Check Amount
- Client Bank Account Number
- Destination Bank Account Number
- Notes

---

## Acceptance Criteria

- Administrators can add payment details to a receipt.
- Payment details are linked to the corresponding receipt.
- Required fields are validated based on the selected payment method.
- Payment information is stored successfully.
- Optional notes can be added.
- Appropriate validation messages are displayed when required information is missing.
